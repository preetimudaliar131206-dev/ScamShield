# 🛡️ ScamShield – AI Scam Detection Platform

ScamShield is an AI-powered web application that helps users detect **phishing emails, scam messages, and suspicious links**. The platform analyzes text, URLs, and screenshots to warn users about potential fraud.

## 🚀 Features

* 📧 **Email Scam Detection** – Analyze emails to detect phishing attempts
* 💬 **Message Analyzer** – Check SMS or text messages for scam patterns
* 🔗 **Link Scanner** – Identify suspicious or phishing URLs
* 🖼 **Screenshot Detection** – Upload screenshots and scan text using OCR
* ⚡ **Risk Classification** – Results labeled as **Safe, Suspicious, or Scam**

## 🏗 Tech Stack

* **Frontend:** html, Next.js, React
* **Backend:** Node.js, Express.js
* **AI Tools:** NLP detection, Regex analysis, Tesseract.js OCR
* **APIs:** Gmail API, Google OAuth*
