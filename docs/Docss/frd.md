FRD – FRONTEND REQUIREMENTS DOCUMENT

Application Name: AI Scam Shield

1. Frontend Overview

The frontend provides a simple, intuitive interface that allows users to monitor scam alerts and manually scan suspicious communications.

The UI prioritizes clarity, fast navigation, and visual risk indicators.

2. Navigation Structure

Main Navigation Sections

• Dashboard
• Manual Detection
• Reported Scams
• Safety Tips

3. Dashboard Page

Displays:

• Protection Status (Real-Time Protection Active)
• Total Scans
• Scams Detected
• Recent Alerts

Color Indicators

Green – Safe
Yellow – Suspicious
Red – Scam Detected

4. Manual Detection Page

Tabs:

SMS Scanner
User pastes suspicious message text.

Call Scanner
User enters phone number.

Email Scanner
User pastes email content or suspicious link.

Screenshot Scanner
User uploads image → OCR extracts text → analysis performed.

5. Detection Result UI

Displays:

• Scam probability score
• Risk meter visualization
• Explanation of suspicious patterns
• Suggested safety action

6. Safety Tips Page

Displays rotating awareness tips such as:

“Banks never ask for OTP through SMS or phone.”

7. Reported Scams Page

Shows:

• Common scam types
• Frequently used scam keywords
• Scam pattern examples
